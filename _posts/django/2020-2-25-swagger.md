django REST의 swagger를 사용해 봅시다.



## 설치

```bash
pip install django-rest-swagger
```

설치한 뒤 django settings.py의 INSTALLED_APPS에 아래와 같이 추가합니다.

```python
INSTALLED_APPS = [
    ...
    'rest_framework_swagger',
    ...
]
```
